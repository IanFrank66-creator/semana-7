1. ¿Qué es un array en programación?
Respuesta: B. Es una infraestructura que permite almacenar varios datos en una lista 
2. ¿Qué método HTTP se utiliza normalmente para consultar datos existentes?
Respuesta: B. GET
3. ¿Qué método HTTP se usa normalmente para agregar información nueva?
Respuesta: B. POST
4. ¿Cuál es el propósito de una API en una aplicación?
Respuesta: B. Permitir que diferentes sistemas o aplicaciones intercambien información.
